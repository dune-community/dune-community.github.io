Detailed Description

template<class RealGridViewImp>
class Dune::XT::Grid::internal::PeriodicIntersectionIterator< RealGridViewImp >

IntersectionIterator for PeriodicGridView.

PeriodicIntersectionIterator is derived from the IntersectionIterator of the underlying GridView and behaves exactly like the underlying IntersectionIterator except that it returns a PeriodicIntersection in its operator* and operator-> methods.
