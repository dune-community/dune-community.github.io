Detailed Description

template<class GridView, int codim>
class Dune::RangedPartitioning< GridView, codim >::Partition

Partition type for ranged partitioning.

This implements the interface for entity ranges. It differs from IteratorEntityRange in that it is TBB-splittable and thus needs to hold a reference to the partitioning.
