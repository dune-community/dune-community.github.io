Detailed Description

template<class GridViewImp>
class Dune::XT::Grid::ApplyOn::WhichIntersection< GridViewImp >

Interface for functors to select on which intersections to apply.

Note
When deriving from this class, one can use the
See Also
internal::WhichIntersectionBase to provide a generic implementation of WhichIntersection::copy in two circumstances: (i) if one requires a BoundaryInfo, derive from internal::WhichIntersectionBase<..., true>,
DirichletIntersections; (ii) if the derived class does not have any members, derive from internal::WhichIntersectionBase<...>,
InnerIntersections. In all other circumstances, one has to manually implement copy,
FilteredIntersections.
