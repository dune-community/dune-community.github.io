Detailed Description

template<class GridViewType, int codim = 0>
class Dune::XT::Grid::EntityInlevelSearch< GridViewType, codim >

Provides a facility to search a given gridview for entities with arbitrary codim that contain a set of points. The search is "in-level", meaning no grid hierarchy is used in the search. The search position iterator on the grid persists between searches, reducing complexity of repeated searches on the grid.

Attention
This makes it inherently not thread safe
