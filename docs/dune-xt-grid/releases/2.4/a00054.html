Detailed Description

template<class GridViewImp>
class Dune::XT::Grid::ApplyOn::InnerIntersectionsPrimally< GridViewImp >

Selects each inner intersection only once.

To decide if this in an inner intersection,

intersection.neighbor() && !intersection.boundary()

is used, and true is returned, if the index of the inside() entity is smaller than the index of the outside() entity.
