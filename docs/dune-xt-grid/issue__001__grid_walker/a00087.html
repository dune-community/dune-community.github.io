Detailed Description

template<class RealGridViewImp, bool use_less_memory>
class Dune::XT::Grid::internal::PeriodicIndexSet< RealGridViewImp, use_less_memory >

IndexSet for PeriodicGridView.

PeriodicIndexSet is derived from the IndexSet of the underlying GridView. Other than the IndexSet, PeriodicIndexSet returns the same index for entities that are periodically equivalent, i.e. entities on periodic boundaries that are regarded as the same entity in the periodic setting. Consequently, the PeriodicIndexSet is usually smaller than the IndexSet and the size(...) methods return lower values than the corresponding methods of the non-periodic IndexSet.
