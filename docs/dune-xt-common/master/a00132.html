Detailed Description

template<class VecType>
struct Dune::XT::Common::VectorAbstraction< VecType >

Traits to statically extract information of a (mathematical) vector.

If you want your vector class to benefit from the operators defined in this header you have to manually specify a specialization of this class in your code with is_vector defined to true and the appropriate static methods and members (see the specializations below).
