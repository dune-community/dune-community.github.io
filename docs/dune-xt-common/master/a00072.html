Detailed Description

template<class MatType>
struct Dune::XT::Common::MatrixAbstraction< MatType >

Traits to statically extract the information of a (mathematical) matrix.

If you want your matrix class to benefit from the operators defined in this header you have to manually specify a specialization of this class in your code with is_matrix defined to true and an appropriate static methods and members (see the specializations below).
