Definition: tuple.hh:28

extracts types of the elements of a Dune::tuple and creates a new Dune::tuple with these extracted types.

first argument: the tuple type second argument: the typedef of the elements of the tuple

example: Let two classes be given:

class A
{
public:
typedef int MyFancyType;
};
class B
{
public:
typedef int MyFancyType;
};

Define a tuple, i.e.

typedef Dune::tuple< A, B, B >
MyTuple;

Instead of writing

typedef Dune::tuple< typename tuple_element<0,MyTuple>::type::MyFancyType,
typename tuple_element<1,MyTuple>::type::MyFancyType,
typename tuple_element<2,MyTuple>::type::MyFancyType >
MySubTupleType;

just write

typedef TUPLE_TYPEDEFS_2_TUPLE( MyTuple, MyFancyType )
MySubTupleType;

without (general) restriction to the size of the tuple MyTuple. Enjoy it!
