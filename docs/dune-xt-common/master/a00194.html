#define DXTC_has_static_member (   mmbr)    DXTC_has_static_member_helper_##mmbr

Macro to statically check a class or struct for a static member.

To check if a class or struct Foo has a static member bar, put this code somewhere, where a generated helper struct may be defined (obviously only once for each member name!, note the trailing ;!):

DXTC_has_static_member_initialize_once(bar);

You can then check for bar (this will give you a static const bool):

DXTC_has_static_member(bar)< Foo >::value

#define DXTC_has_typedef (   tpdf)    DXTC_has_typedef_helper_##tpdf

Macro to statically check a type for a typedef.

To check if a type Foo defines a typedef Bar, put this code somewhere, where a generated helper struct may be defined (obviously only once for each typedef name!, note the trailing ;!):

DXTC_has_typedef_initialize_once(Bar);

You can then check for Bar (this will give you a static const bool):

DXTC_has_typedef(Bar)< Foo >::value
