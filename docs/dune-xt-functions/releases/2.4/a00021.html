Detailed Description

template<class E, class D, size_t d, class R, size_t r, size_t rC = 1>
class Dune::XT::Functions::FunctionsFactory< E, D, d, R, r, rC >

Note
If you want to add a new function FooBar, do the following: provide a definition that is available for all template arguments, like:
template< class E, class D, size_t d, class R, size_t r, size_t rC = 1 >
class FooBar
: public LocalizableFunctionInterface< E, D, d, R, r, rC >
{
FooBar() { static_assert(AlwaysFalse< E >::value, "Not available for these dimensions!"); }
};
Every specialization that can be provided by the provider then has to define:
static const bool available = true;
In addition you have to add the appropriate include in this file (of course) and the appropriate type below (just like the rest, should be obvious).
