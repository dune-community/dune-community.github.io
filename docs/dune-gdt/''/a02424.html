Detailed Description

template<class GridLayerImp, class SourceImp, class RangeImp>
class Dune::GDT::LagrangeProlongationLocalizableOperator< GridLayerImp, SourceImp, RangeImp >

Carries out a prolongation (in a localized manner) using a lagrange projection.

This is done by reinterpreting the source on the range grid layer and applying a LagrangeProjectionLocalizableOperator.
