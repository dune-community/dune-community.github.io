Detailed Description

template<class BasisfunctionImp, class GridLayerImp, class U>
class Dune::GDT::EntropyBasedLocalFlux< BasisfunctionImp, GridLayerImp, U >

Analytical flux {f}({u}) = < {m} G_{{}({u})} >, for the notation see Alldredge, Hauck, O'Leary, Tits, "Adaptive change of basis in entropy-based moment closures for linear kinetic equations"
