template<class E, class D, size_t d, class R, size_t r, size_t rC>
class Dune::GDT::AnalyticalFluxInterface< E, D, d, R, r, rC, true >

Analytical flux f for problem of the form delta_t u + div f(u,x,t, u) = 0 where u: R^d R^r rC}.
