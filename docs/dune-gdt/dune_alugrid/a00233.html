Detailed Description

template<class GridViewImp, class SourceImp, class RangeImp>
class Dune::GDT::LagrangeProlongationLocalizableOperator< GridViewImp, SourceImp, RangeImp >

Carries out a prolongation (in a localized manner) using a lagrange projection.

This is done by reinterpreting the source on the range grid view and applying a LagrangeProjectionLocalizableOperator.
