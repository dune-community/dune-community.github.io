Detailed Description

template<class AnalyticalFluxImp>
class Dune::GDT::GodunovLocalNumericalCouplingFlux< AnalyticalFluxImp >

Lax-Friedrichs flux evaluation for inner intersections and periodic boundary intersections.

The Lax-Friedrichs flux is an approximation to the integral {S_{ij}} {F}({u}) {n}_{ij}, where S_{ij} is the intersection between the entities i and j, {F}({u}) is the analytical flux (evaluated at {u}) and {n}_{ij} is the unit outer normal of S_{ij}. The Lax-Friedrichs flux takes the form {g}_{ij}^{LF}({u}_i, {u}_j) = {S_{ij}} {1}{2}({F}({u}_i) + {F}({u}_j) {n}_{ij}

  • {1}{ {ij}} ({u}_j - {u}_i), where is the number of neighbors (i.e. intersections) of the entity i and lambda_{ij} is a local constant fulfilling {ij} {{u}} ({F}({u} {n}_{ij})^ 1. The integration is done numerically and implemented in the LocalCouplingFvOperator. This class implements the evaluation of the integrand. As we are restricting ourselves to axis-parallel cubic grids, only one component of {n}_{ij} is non-zero, denote this component by k. Then the Lax-Friedrichs flux evaluation reduces to {1}{2}({f}^k({u}_i) + {f}^k({u}_j) n_{ij,k}
  • {1}{ {ij}} ({u}_j - {u}_i), where {f}^k is the k-th column of the analytical flux. For the classical Lax-Friedrichs flux, {ij} is chosen as dt/dx_i, where dt is the current time step length and dx_i is the width of entity i. This fulfills the equation above as long as the CFL condition is fulfilled. The local Lax-Friedrichs flux can be chosen by setting
    Parameters
    use_localto true, here {ij} is chosen as the inverse of the maximal eigenvalue of {f}^k({u}_i) and {f}^k({u}_j). In this case, you should also specify whether your analytical flux is linear by setting
    is_linear,whichavoids recalculating the eigenvalues on every intersection in the linear case. You can also provide a user-defined
    lambdathat is used as {ij} on all intersections. You need to set use_local to false, otherwise lambda will not be used.
