template<class Traits, class D, size_t d, class R, size_t r, size_t rC = 1>
class Dune::GDT::BaseFunctionSetInterface< Traits, D, d, R, r, rC >

The purpose of this interface is just to be used for template matching and to allow for access to the backend. All other functionality is enforced by XT::Functions::LocalfunctionSetInterface.
